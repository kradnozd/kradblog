Tải và cài đặt Open Broadcast Software

Tải bản Classic hoặc Studio tại https://obsproject.com/download

Về cơ bản đều giống nhau cả, dùng bản nào cũng được. Bản Studio hỗ trợ nhiều tính năng live chuyên nghiệp hơn và hỗ trợ multi platform. Trong ví dụ này dùng bản Studio.

Lấy key live stream

Bằng cách nhấn vào nút màu xanh to đùng ở phía trên, một cửa sổ popup sẽ hiện ra. Nếu browser chặn popup thì phải allow. Chọn nơi muốn share live stream rồi bấm Next, thường thì share lên timeline của bản thân nên bấm Next luôn khỏi cần nghĩ.

Ở popup tiếp theo điền Video title, tags, vài câu status giới thiệu,... Phần quan trọng nhất là ô Stream key ở bên phải, copy stream key ở ô này rồi bật OBS lên.

Vào Settings / Stream (Hoặc Setting / Broadcast), chọn Service là Facebook Live rồi điền stream key vào ô Stream key.

Nếu không có lựa chọn Facebook Live trong Service thì chọn Stream type là Custom Streaming Service, rồi copy thêm ô Server URL ở popup điền vào phần URL.

Chọn OK để lưu Setting rồi nhấn nút Start Streaming. Đợi khoảng vài giây thì ở bên popup Facebook Live Stream sẽ nhận được tín hiệu và có preview video, nhấn nút Go Live để bắt đầu stream. Lưu ý rằng nếu đang dùng Adblock thì có thể preview video sẽ không hiện lên được, lúc này phải đặt ngoại lệ cho domain Facebook trong Adblock để có thể stream.
